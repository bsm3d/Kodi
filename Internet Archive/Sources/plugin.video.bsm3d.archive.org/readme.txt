    INTERNET ARCHIVE BROWSER KODI ADDON - README

PROJECT DETAILS
-----------------------------------------------------
Name            : Internet Archive Browser
Version         : 1.0.0
Kodi            : Kodi 21 (Omega) and later
Author          : BSM3D

QUICK OVERVIEW
-----------------------------------------------------
Explore the world's largest digital library through 
your Kodi media center. Access an extensive collection 
of videos, audio, classic films, and more.

KEY FEATURES
-----------------------------------------------------
* Video and Audio Collection Browsing
* Advanced Search Functionality
* Multiple Content Categories
* French Content Sections
* Favorites Management
* Playback History
* Comprehensive Content Filtering

PREREQUISITES
-----------------------------------------------------
* Kodi 21 (Omega) or newer
* Active Internet Connection
* Optional: script.module.requests

INSTALLATION METHODS
-----------------------------------------------------
Method 1: Kodi Repository
1. Open Kodi
2. Navigate to Add-ons > Download
3. Search for "Internet Archive Browser"
4. Click Install

Method 2: Manual Installation
1. Download addon ZIP file
2. Open Kodi > Settings > Add-ons
3. Choose "Install from ZIP file"
4. Select downloaded addon file

CONFIGURATION
-----------------------------------------------------
1. Launch Internet Archive Browser Addon
2. Configure settings:
   - Content Filtering Options
   - Cache Duration
   - Debug Mode
   - Custom Categories

USAGE OPTIONS
-----------------------------------------------------
Main Menu:
* Browse Audio Collections

* French Content

* Search

CONTENT FILTERING

TROUBLESHOOTING
